## What todo in all prompts

1. Check .venv exists, if do not run `sh init_py.sh`
2. ALWAYS follow the coding stardands defined in `agents/0_CODING_STANDARD`
3. ALWAYS update the `./agents/MEMORY.md` with a detailed description of what was made to develop the solution.
4. ALWAYS run unit tests by the end.

## Important files

| File                          | Description                                                                                   |
| ----------------------------- | --------------------------------------------------------------------------------------------- |
| `AGENT.md`                    | Instructions that the Agents have to follow                                                   |
| `agents/0_CODING_STANDARD.md` | Coding standar that have to be followed                                                       |
| `README.md`                   | File with the context about the project                                                       |
| `agents/MEMORY.md`            | Work as a MEMORY BANK. It contains everything made by agents. Should be updated all the time. |
